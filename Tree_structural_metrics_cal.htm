Tree metrics calculation using the iTSme package:

Author: Arunima Singh, Louise Terryn

Installation:

You can install the development version of ITSMe from ITSMe in R-Studio:

# install.packages("devtools")

devtools::install_github("lmterryn/ITSMe", build_vignettes = TRUE)

The tree metrics focused here are DBH (Diameter at breast height), DBH (above buttress), crown volume, and CPA (Crown Projected Area)

1.       DBH (Diameter at Breast Height)

# Read the point cloud file from the specified path to the tree point cloud file

pc_tree <- read_tree_pc(path = "path/to/point/cloud.txt")

# Use dbh_pc function with default parameters and plot the fit

out_dbh <- dbh_pc(pc = pc_tree, plot = TRUE)

 

2.       DBH (above buttress)

# Use dbh_pc function with default parameters and plot the fit

out_dbh <- dab_pc (pc = pc_tree, plot = TRUE)

3.       Tree height

Tree_height<-tree_height_pc(pc = pc_tree, plot = TRUE)

4.       Crown Classification:

Basis for crown metrics

Crown points = all points of the tree except for the stem points below the first branch

Classify_crown<-classify_crown_pc(pc = pc_tree, plot = TRUE)

a.       CPA (Crown projected area)

                                                                              i.       After the classification of the crown:

The area of the vertical projection of the tree crown

From point cloud: area of the concave hull fitted to the XY coordinates of the crown points

Projected_area <- projected_area_pc(pc = pc_tree, plot = TRUE)

b.      Crown Volume

                                                                             i.        From point cloud: volume of the alpha shape fitted to the crown points

Crown_volume <- alpha_volume_pc(pc = pc_tree, plot = TRUE)

 

The overall summary calculation with basic structural metrics

Calculating a summary data.frame with the basic structural metrics (tree position, dbh, dab, tree height, projected area, 3D alpha volume) that can be obtained from individual tree point clouds for all point clouds in a specific folder:

basic_summary <- summary_basic_pointcloud_metrics(

  PCs_path = "path/to/point/cloud/folder/",

  OUT_path = "path/to/output/folder/")

 

Tree metrics calculation using 3DFin (CloudCompare) plugin:

1.       Installation of CloudCompare:

You need to have latest version of CloudCompare v2.14.alpha follow the link cloudcompare, you should download the indicated one in the picture below:

 

Download it and follow the installer instructions, you need to check the 3DFin plugin while installation of the software.

 

2.       Process the point cloud with 3DFin

 

Then open CloudCompare after successful installation, and you can see the plugin as shown in the figure below:

 

 

Open the point cloud file, it runs on plot, it has 3 options Basic, Advanced and Expert, using this alternative options you can have more control over over outputs otherwise use basic with default setup. You can explore and play with the advanced and expert set-up later as well.

With Basic default setup, just select you point cloud  and choose the output directory and Run.

 

 

3.       Output:

The output (.xlsx or .txt file) will be saved in the chosen directory, and other outputs (DTM, Stems in stripe, Highest points, fitted sections, Axes, and Tree locator) opened automatically in CloudCompare after the process is finished :
